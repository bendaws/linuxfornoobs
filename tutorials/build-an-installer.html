Build a Linux Bootable Installer

A complete guide for beginners. Read the Quick-Start guide.

Now that you have chosen your distro, it's time to use the .iso image to make a USB that you can boot into (in other words, it is a temporary system that installs Linux). You must have these items in order to install Linux:

  • A USB Flash Drive, CD / DVD, or another storage device with atleast 32GB of storage (refered to as a medium)
  • An .iso file downloaded from your distro of choice's website
  • Balena Etcher for creating a USB

    • Creating a bootable medium is much easier than you may expect. While Linux does appear to be very techy and technical (it is, in most cases), installing it is much simpler than it was.

    A small, lightweight application called Etcher will create the installer for you. Once you have downloaded the application, you can select "Flash from File", choose your .iso image, select your USB, and select "Flash".

    It will make an installer for you.
